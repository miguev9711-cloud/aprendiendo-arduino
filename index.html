💡 Conceptos esenciales

Arduino es una plataforma de electrónica open-source basada en hardware y software fáciles de usar. Un programa en Arduino tiene dos funciones principales:

  • setup() → se ejecuta una sola vez al inicio. Configura pines, comunicación serie, etc.
  • loop() → se repite infinitamente. Aquí va la lógica principal.
// Ejemplo: Encender un LED por 1 segundo, apagarlo 1 segundo
void setup() {
  pinMode(13, OUTPUT); // Configurar pin 13 como salida
}
void loop() {
  digitalWrite(13, HIGH); // Encender LED
  delay(1000); // Esperar 1000 ms
  digitalWrite(13, LOW); // Apagar LED
  delay(1000);
}

✅ digitalWrite(pin, HIGH/LOW) → envía voltaje (5V) o 0V.
✅ delay(ms) → pausa el programa durante milisegundos.

📝 Ejercicio 1: Parpadeo de LED

void setup() {

void loop() {

📝 Ejercicio 2: Control con pulsador

Teoría: digitalRead(pin) lee el estado de un botón (HIGH o LOW). Completa el siguiente programa para que el LED se encienda solo cuando el pulsador esté presionado (pulsador conectado al pin 2).

int ledPin = 13;
int buttonPin = 2;
void setup() {
  pinMode(ledPin, OUTPUT);
  pinMode(buttonPin, INPUT);
}
void loop() {
  int estado = digitalRead(buttonPin);
  if (estado == HIGH) {
    digitalWrite(ledPin, __________);
  } else {
    digitalWrite(ledPin, LOW);
  }
}
